⚙️ Setup Google Sheets (Sekali Saja)
1
Buat Google Sheets baru
Buka sheets.google.com → Buat spreadsheet baru → Beri nama "Donasi Bahula Tani"
2
Buat header kolom di baris pertama
Isi kolom A–F:
Waktu   Nama   Nominal   Tanaman   Lokasi   Pesan
3
Bagikan ke publik (view only)
Klik Share → Anyone with the link → Viewer → Done
4
Paste Sheet ID di sini
Dari URL: docs.google.com/spreadsheets/d/[SHEET_ID]/edit
